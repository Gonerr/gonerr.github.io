FullStack and Web developer
---------------------------

It was my computer science classes that inspired me to become a programmer - I like thinking about problems and creating something that can be used. And I've always been most attracted to creative things: web applications, HTML pages, presentations, and so on. That's how I left my hometown and enrolled in Applied Informatics at SPbPU.

I spent four years there studying: I created various programs in C#, C++, Python, Java, and PHP, developed large databases in PostgreSQL, and solved numerous complex mathematical algorithms. However, I was most interested in programming in JavaScript, so I created my first large projects in this language. Not all projects are included in this GitHub, but you can see the ones that have been preserved.

During my studies, I managed to work at various companies: I worked as a tutor for three years, teaching children to program in Python, which helped me develop my soft skills; I worked for a year at a law firm as a web programmer and system administrator for Bitrix24, automating many business processes, creating WPF web applications for simplified transaction processing, and websites for working with data and Excel on JS.

So I came to the conclusion that React was my favorite library - I created mobile apps on React Native and simple websites with MUI and Redux. I wrote servers on ASP .NET and NodeJS. As a result, I changed jobs and worked as a web programmer at a construction company, where I created work applications for employees, worked with internal CRM, created plugins in C#, and supported a legacy project using Azure - I developed full-fledged components and organized data storage in the database. Now I am striving to develop further by learning new things: doing my own pet projects and studying the intricacies of JavaScript.
